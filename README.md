# 🎮 2048: Python 

"A Tile-merging game built with the Pygame prerequisite".

## 🖥️ Operating Systems Supported: 

Windows 7/8/8.1 (Provided you use Python 3.8-3.12, and you check your CPU architecture)
Windows 10/11
macOS 10.15 Catalina and later(including Sequoia provided you have Python 3.8-3.13 AND ONLY 64-bit CPU Architecture with Intel or Apple Silicon)
All Linux with Python 3.6 or later installed


## 🚀 Quick Start

Get the game up and running locally in less than two minutes.

### Prerequisites
* **Python 3.9 or higher** installed on your system.

### Installation & Run Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd Blastmaster-Fixup-and-Kleanup-Crew/2048-Python
   ```

2. **Set up a virtual environment (Recommended):**
   * **Windows:**
     ```bash
     python -m venv venv
     .\venv\Scripts\activate
     ```
   * **macOS/Linux:**
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Launch the game:**
   ```bash
   python main.py
   ```

## 🎮 How to Play

Merge tiles of the same value to get to the 2048 tile

*   **Move:** `W`, `A`, `S`, `D` or Arrow Keys to mmove and merge tiles


**Objective:** Merge tiles of the same value to get to the 2048 tile to win.
